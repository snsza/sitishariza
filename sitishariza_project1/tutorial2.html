Based on my previous javaScript video tutorial, I have created a simple order online page which is "Safiyya Photobook Online Order Form" by using a full coding in notepad.   • JavaScript Functions



    • I created a function named function check() inside the script tag. The function that I have created is used to validate every value from the form inside the body. var is used to declare any variables in javaScript, eventhough the variable is a string, double or any types. "document.form1.name.value" is used to retrieve the value from the input data with the given name name in the form named form1. Then, I used selection structure, "if", to give conditions to the variables. For example, if the input data is blank or if the user input data not in formatted value, a message will popup on the screen to alert the user to fill in the form.



    function total() is used to calculate the total price for every purchased from the users. I also used the selection structure to give the condition to the variables declared. The prompt() is used to prompt the user to input some data and it will be open in dialog box. If the user enter "y", a dialog box will appear on screen with the total price of the purchased, if the user enter "n", a message will appear in dialog box. alert is used to open the dialog box.

  • Form Event



    • onsubmit event is used so that an event will occurs when a form is submitted. In my html coding, I used this event to validate my form. the event will be initialized to function check() that I have created.

  • Mouse Event



    • onclick event is used so that an event will occurs when the user clicks on the "ORDER" button. The event will be initialized to function total() that will calculate the total price for every purchased.
