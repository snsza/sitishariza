Based on my previous video tutorial, I have created a simple calculator page by using a full coding in notepad. In my html with javaScript coding, I used a function and an event to create the calculator page. Below are the screenshots of my html with javaScript coding :

  • Full Coding



  • JavaScript Function



    • I created a function named function displayNum(n1) inside a script element which is script tag. The function that I have created is used to read every value from the form inside the body that being called by the parameter, n1. Then, it will display every value in the textbox named text1.

  • JavaScript Form



    • Inside the form, all the input types are in button so that it will display like a calculator in the webpage. Besides that, I have also include a CSS in my html coding to make all the width and height of the calculator's buttons are in the same size.

  • Mouse Event



    • I used mouse event which is onclick event so that an event will occurs when the users clicks on an element, which is the button. Then, the onclick event will be initialized to the function that I have created with a parameter, onclick="displayNum(btn7.value);". The parameter consists of the name of the input form as well as its value.



    Inside the form, there is an input type of button that I have set it to has a specific function of equal to or sum in the calculator. Therefore, I used an onclick event that be initialized with the value in the textbox equal to an eval function, onclick="text1.value = eval(text1.value);". The eval function is used to evaluate/execute expressions or statements. So, I put parameter in the eval function that consists of the value in the textbox, so that it will execute all the expressions in the textbox.

  • Reset



    • There is also a CLEAR or reset button on my calculator so that the users are able to reset all the input. I used an input type of reset, input type="reset".
